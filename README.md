## API publica para consulta de CNPJ em NextJS utilizando o cnpj.ws 

<a href="https://nextjs-cnpjws-rkij.vercel.app/" target="_blank">Demo</a>


## Instalação

```shell
npm i
``

## Iniciar

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Ferramentas utilizada no projeto

- ⚡️ Next.js 13
- ✨ TypeScript
- 💨 Tailwind CSS
- 📏 ESLint
- 💖 Prettier
- 🤖 chat-gpt
